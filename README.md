## 概念解析

#### ORM: Object-Relational-Mapping

对象关系映射: 在关系型数据库和业务实体类之间建立映射关系, 其目的是让开发者在业务开发中:

[√] 集中精力在业务实体类设计和简单的操作接口上

[x] 省去编写对繁琐的数据库读写操作语句
